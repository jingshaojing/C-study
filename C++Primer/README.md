**1.8.嵌套注释的输出合法性**

注释界定符的嵌套是以 /* 开始，以*/结束，因此在这两者之外的输出必须符合语法规则。如下面：

```c++
cout << /*"*/"*/" << endl;
cout <</* "*/" /* "/*" */ << endl;
```

报错：

```
error C2001: 常量中有换行符
1>E:\c++\Cppstudy\C++Primer\ch01\Output nesting\practice1.18.cpp(11,2): error C2146: 语法错误: 缺少“;”(在标识符“cout”的前面)
```

在第一行程序中再加一个双引号即可解决。
